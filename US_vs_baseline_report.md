# 美股標的實驗報告:事件+市場 融合模型 vs. 市場-only 基準

> 資料來源:`US_vs_baseline.csv`(由 `evaluate_matrix.py` 產出)
> 標的:`TSM` / `^SOX` / `^NDX` / `^GSPC`(美股)+ `2330.TW`(台股對照)
> 模型:`gated_mlp`,`--presidential-terms-only`,`--hold 1`,`--auto-trade-threshold`,`--trade-mode long_short`,80 epochs
> 事件特徵已依**各標的所在市場時區**重新分桶/歸日(Layer 1 + Layer 2);美股/台股各用 `_us` / `_tw` 事件 CSV。

---

## 0. 實驗設計回顧(三個關鍵詞)

- **full**(`full_event_market_gated_mlp`):市場特徵 **+** 川普事件/regime 特徵的完整融合模型。
- **baseline**(`market_only_gated_mlp`):**消融控制組** —— 把所有事件/regime/交互特徵換成單一零欄位,只留市場特徵。
- **`d_<metric>`** = `full − baseline`,即「**加入事件後的淨變化**」,也就是這個研究的核心量:**事件加成(event edge)**。
  - `d_ > 0` → 事件有幫助;`d_ < 0` → 事件是雜訊、反而拖累。

---

## 1. 各指標定義(如何讀這張表)

模型本身是**二元方向分類**(明日 close→close 報酬 > 0 為「漲」),交易層再把機率轉成 LONG/SHORT/NEUTRAL。指標分兩類:

### 1.1 分類品質指標(看模型「方向猜得準不準」)

| 指標 | 定義 | 怎麼讀 |
| :-- | :-- | :-- |
| **accuracy** | 方向猜對的比例(對的天數 / 總天數) | 0.5 = 擲銅板;>0.5 才有方向訊號 |
| **macro_f1** | 漲、跌兩類各自 F1 後取平均(不被多數類灌水) | 對類別不平衡更誠實;>0.5 佳 |
| **precision** | 模型說「漲」之中真的漲的比例 | 高 = 出手喊漲時較可信 |
| **recall** | 真的漲的日子裡被模型抓到的比例 | 低 = 漏抓很多上漲日(偏保守) |
| **auc** | ROC-AUC,模型「機率排序」把漲日排在跌日前面的能力 | 0.5 = 無排序力;**<0.5 = 排序方向是反的(比亂猜還差)** |

### 1.2 策略獲利指標(看「照訊號下單能不能賺」)

| 指標 | 定義 | 怎麼讀 |
| :-- | :-- | :-- |
| **cumret** | 測試期照 LONG/SHORT 訊號累積的報酬(複利) | **診斷用,未計手續費/滑價/借券** |
| **sharpe** | 年化風險調整後報酬(`√252 · 平均/標準差`) | 越高越穩;樣本太少時無意義 |
| **trades** | 測試期實際下單(非 NEUTRAL)的次數 | **太少(個位數)→ 報酬/Sharpe 不可信** |

> ⚠️ 兩個鐵則:**(a) accuracy ≠ 獲利**;**(b) trades 很小的 cumret/sharpe 要直接忽略**(見 ^NDX)。

## 3. 分析

### 發現 1 — 「可預測性」:台股遠勝任何美股標的
full 模型在 **2330.TW** 的 accuracy 0.626、AUC 0.725,明顯找到了真實的方向結構;反觀美股標的全部 ≤ 0.55,而且兩大盤的 **AUC 還低於 0.5**(^GSPC 0.480、^NDX 0.461)—— 代表模型在美股大盤上的機率排序近乎隨機、甚至方向是反的。**深度融合模型只在台股本地市場學到東西,在美股大盤等於沒有訊號。** 這符合 US.md plan 3.3 的預期(大盤效率高、最難擠出加成)。

### 發現 2 — 「事件加成」集中在半導體相關標的
依 d_accuracy 排序:**^SOX (+0.068) > TSM (+0.027) > 2330.TW (+0.015) > ^GSPC (−0.033) > ^NDX (−0.041)**。
- **^SOX 是最大贏家**:事件把 accuracy 從 0.457(基準甚至低於擲銅板)拉到 0.525,cumret 從 −0.168 翻到 +0.008(d_cumret +0.176)。對晶片/關稅/中國言論最敏感的費半,確實最吃得到事件訊號 —— **完全命中 plan 的頭號預期**。
- **兩大盤(^GSPC/^NDX)事件是負加成**:加入事件後 accuracy、AUC、macro_f1 全部下降,事件對效率市場只是雜訊。

### 發現 3 — 頭條對照 TSM vs 2330.TW(同一家公司、不同掛牌)
| | d_accuracy | d_auc | d_macro_f1 |
| :-- | --: | --: | --: |
| TSM(ADR) | **+0.027** | +0.023 | +0.023 |
| 2330.TW(本地) | +0.015 | **+0.028** | +0.014 |

兩者的事件加成都**很小且方向一致為正**:accuracy 上 TSM 略大、AUC 上 2330.TW 略大 —— 基本是**平手**。
- 這 **沒有支持** US.md plan 3.4「本地落後市場(2330.TW)事件加成明顯大於 ADR(TSM)」的押注;
- 也**沒有強力支持**你的假設「美股反應更快更大 → 對隔日更可預測」。TSM 的 accuracy 加成雖略高,但幅度(±0.01–0.03)落在單次跑的雜訊範圍內,不足以下定論。

### 發現 4 — 獲利由「市場」而非「事件」主導,且 accuracy ≠ 獲利
- 2330.TW 的 cumret 2.07 很驚人,但**主要來自 market 基準(2.37)**;加入事件後 cumret 反而略降(d_cumret −0.302)。台股的高報酬是強動能/regime 訊號,不是政治輿情。
- 美股的 cumret 全部貼近 0;唯一事件改善交易報酬的是 **^SOX(d_cumret +0.176)**。
- 多個標的出現「accuracy 上升、cumret 下降」(如 TSM:d_accuracy +0.027 但 d_cumret −0.217)—— 再次印證**方向準度與交易績效脫鉤**。

### 發現 5 — full 模型變得「惜售」,交易數全面崩跌
所有標的的 **d_trades 都是大幅負值**(−45 ~ −105):auto-threshold 在 full 模型上挑了更高的 edge 門檻,出手次數銳減。
- **^NDX full 只下單 2 筆** → 它的 sharpe 1.41 / cumret 0.019 **統計上無意義,必須忽略**。
- TSM 從 150 筆掉到 45 筆、2330.TW 從 189 掉到 138。事件特徵讓模型更挑、更常 NEUTRAL 觀望。

## 4. 結論與對照假設

| 假設來源 | 主張 | 本實驗結果 |
| :-- | :-- | :-- |
| US.md plan 3.3 | 加成排序 ^SOX 最高、大盤 ~0 或負 | ✅ **成立**(^SOX 最高、^GSPC/^NDX 負) |
| US.md plan 3.4 | 2330.TW 事件加成 > TSM | ❌ **未成立**(兩者平手,TSM 的 accuracy 加成甚至略大) |
| 你的假設 | 美股反應更快更大 → 對隔日更可預測 | ⚠️ **部分** —— 事件確實對美股半導體(^SOX/TSM)幫助較大,但**美股整體可預測性遠低於台股,且當日大反應沒有轉成隔日可預測性**(正如先前提醒:反應當天就被 close[D] 吸收) |

**一句話**:事件訊號真正有用的地方是**對政策最敏感的半導體**(^SOX 最明顯,TSM 次之),對**美股大盤是純雜訊**;而**台股本地市場最可預測**,但那主要是市場/動能效應,不是政治輿情。

---

## 5. 重要警語與後續建議

- **這些 cumret/sharpe 是診斷值**:未計手續費、滑價、借券、部位大小,不等於真實可交易績效。
- **^NDX(2 筆)的獲利指標請當作不存在**。
- **單次跑、未做多 seed 平均**:±0.01–0.04 的 d_accuracy 落在 run-to-run 雜訊內。只有 **^SOX 的 +0.068 accuracy / +0.176 cumret** 與**兩大盤的負加成**幅度夠大、較可信賴。
- **建議**:
  1. 對 ^SOX、TSM、2330.TW 做 **3–5 個 seed 平均**,確認小幅加成是否穩定為正。
  2. 報告主打 **^SOX = 最乾淨的正面案例**;頭條對照改寫成「同公司兩市場的事件加成皆小且相當」。
  3. 美股大盤(^GSPC/^NDX)可作為「市場效率 → 事件無效」的反面對照寫進報告。
